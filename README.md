# Khash – Personal Trainer (Frontend)

A React single-page website for a personal trainer: **Home, Services, About, Testimonials, Contact, and Book Session**.  
Built as a practice project to learn React, routing, animations, and UI/UX basics.

> **Status**
> - The site is marketing-ready.
> - **Contact form and booking are temporarily disabled** (clearly indicated in the UI) until email/booking is configured.

## Tech Stack
- **React 18** (functional components & hooks)
- **React Router** (client-side routing)
- **Framer Motion** (subtle animations)
- **CSS (mobile-first)** with custom variables
- **React Icons**

---

## Features
- Responsive single-page design
- Booking form with input validation
- Smooth animations using Framer Motion
- Easy navigation with React Router

## Run Locally
To run this project on your machine:

**Prerequisites**
- Node.js 18+ and npm

```bash
# clone the repo
git clone https://github.com/kmodabber/trainer-website-client.git

# go into the project folder
cd trainer-website-client

# install dependencies
npm install

# start the dev server
npm start
