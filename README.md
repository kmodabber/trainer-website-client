# Khash – Personal Trainer (Frontend)

A React single-page website for a personal trainer: **Home, Services, About, Testimonials, Contact, and Book Session**.  
Built as a practice project to learn React, routing, animations, and UI/UX basics.

## Tech Stack
- **React** (CRA / react-scripts)
- **React Router**, **Framer Motion**, **React Icons**
- **CSS modules** (per page)

---

## Features
- Responsive single-page design
- Booking form with input validation
- Smooth animations using Framer Motion
- Easy navigation with React Router

## Future Improvements
- Backend integration to handle bookings and email notifications
- Improved styling with Tailwind or styled-components
- Deployment to Netlify or Vercel for public access
- Add unit tests for components

## Run Locally
To run this project on your machine:

```bash
# clone the repo
git clone https://github.com/kmodabber/trainer-website-client.git

# go into the project folder
cd trainer-website-client

# install dependencies
npm install

# start the dev server
npm start
