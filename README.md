---

---

## Tech Stack
- **React** (CRA / react-scripts)
- **React Router**, **Framer Motion**, **React Icons**
- **CSS modules** (per page)

---

## Features
- Responsive single-page design
- Booking form with input validation
- Smooth animations using Framer Motion
- Easy navigation with React Router

## Run Locally
To run this project on your machine:

```bash
# clone the repo
git clone https://github.com/kmodabber/trainer-website-client.git

# go into the project folder
cd trainer-website-client

# install dependencies
npm install

# start the dev server
npm start
